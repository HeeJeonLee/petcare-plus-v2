## 3️⃣ 핵심 기능 (1/2)

### 1️⃣ AI 맞춤형 펫보험 추천
- Claude AI 기반 개인화 알고리즘
- 반려동물 정보 입력 → 최적 보험 자동 추천
- 이유 설명 포함

### 2️⃣ 8개 보험사 상세 비교표
- **메리츠 펫퍼민트**: 점유율 1위, 제휴병원 많음
- **삼성 위풍댕댕**: 치과 특화, 다견 할인
- **현대 굿앤굿우리펫**: 100% 보장, 가성비
- **DB 펫블리**: 슬개골 특화, 12세까지
- **KB 금쪽같은펫**: MRI/CT 최고, 대형견
- **한화 시그니처**: 2026년 신상품
- **농협 NH가성비굿**: 배상책임 높음
- **롯데 let:click**: 저렴, 간편 가입

비교 항목: 월 보험료, 보장내용, 특징, 가입자 평가

### 3️⃣ AI 챗봇 상담 (24시간)
- Claude API 기반 고급 상담 시스템
- 펫 건강 정보 제공
- 보험 선택 가이드
- 실시간 답변 (전문가 수준)
