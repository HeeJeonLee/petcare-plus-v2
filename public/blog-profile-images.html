사용 방법

  1. 원하는 이미지에서 마우스 오른쪽 클릭 → 이미지로 저장
  2. 네이버 블로그 → 관리 → 기본 설정 → 프로필
  3. 프로필 이미지 변경 → 저장한 이미지 업로드
  4. 블로그 별명: PetCare+ 이희전 추천
SVG 파일이 업로드 안 되면, 이미지를 열고 스크린샷(캡처)해서 PNG로 저장 후 업로드하세요.
